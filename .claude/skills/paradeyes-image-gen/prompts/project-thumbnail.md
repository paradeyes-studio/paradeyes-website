# Template, Project Thumbnail (vignette étude de cas)

## Specifications techniques

- Aspect ratio, 1:1 (carré pour grille) ou 4:5 (vertical pour cards listing)
- Resolution, 1200x1200 ou 1200x1500
- Format de sortie, PNG → WebP optionnel
- Save path, `public/images/projets/{slug}-thumbnail.png`

## Base prompt en anglais

Premium brand campaign visual for a creative communication agency case study.
Composition, centered focal element, minimal background, strong product or concept presence, strategic use of negative space.
Lighting, studio quality, controlled directional light, clean shadows, premium catalog feel.
Color palette, composite gradient backgrounds using ONLY deep green #003135, electric green #57EEA1 in subtle accents, dark bluish #14222E for depth, warm white #FAFAF7 for highlights. Mix two of these for the gradient base.
Texture, optional subtle pattern overlay (fine grid, diagonal hatching) at very low opacity.
Mood, confident, premium, conceptual, brand-focused.
Style references, Vercel project showcase, Linear feature cards, Resend blog hero images, modern SaaS premium visuals.
NO text, NO real-world brand logos visible, NO photographs of identifiable people.
